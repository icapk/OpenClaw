## 核心工作优先级

| 优先级 | 工作 | 说明 |
|--------|------|------|
| 1 | 学习爆款写作 | 每天必须，分析拆解新爆款 |

## 一、学习是核心（每天必须执行）

### 学习内容

| 平台 | 学习重点 |
|------|----------|
| 小红书 | 爆款笔记的标题、封面、内容结构 |
| 公众号 | 爆款文章的观点深度、阅读体验 |

### 学习渠道

1. **竞品账号**：关注同领域头部账号的爆款内容
2. **平台热榜**：小红书热榜、公众号热文
3. **跨平台学习**：微博、知乎、即刻等
4. **专业分析**：新红、千瓜等数据平台

### 学习输出

每次学习后记录：
- 爆款标题模板（收藏10个）
- 内容结构规律（总结1条）
- 可借鉴的技巧（记录3个）

### 学习时间

- 每天4:00开始，先学习30分钟
- 分析2-3篇新爆款
- 提取可复用的技巧

## 二、区分平台运营（关键区别）

### 小红书特点

| 维度 | 特点 |
|------|------|
| 受众 | 年轻女性为主，25-35岁 |
| 内容风格 | 短平快、视觉化、情绪化 |
| 标题 | 15-20字，悬念/数字/情绪 |
| 封面 | 强视觉冲击，大字标题 |
| 正文 | 分点+emoji，段落短 |
| 标签 | 5-8个精准tag |
| 互动 | 收藏率高=干货，评论率=共鸣 |

**小红书爆款公式：**
```
hook标题 + 惊艳封面 + 干货分点 + 闺蜜语气 + 精准标签
```

### 公众号特点

| 维度 | 特点 |
|------|------|
| 受众 | 职场人、创业者、知识分子 |
| 内容风格 | 深度、思考、价值感 |
| 标题 | 不限字数，重观点 |
| 封面 | 简洁大气，不过度设计 |
| 正文 | 深度论述，逻辑清晰 |
| 阅读体验 | 排版重要，段落清晰 |

**公众号爆款公式：**
```
独特观点 + 深度论证 + 金句收尾 + 优质排版
```

### 核心区别

| 维度 | 小红书 | 公众号 |
|------|--------|--------|
| 内容长度 | 300-800字 | 1000-3000字 |
| 深度要求 | 浅但实用 | 深且独特 |
| 视觉要求 | 极高 | 中等 |
| 更新频率 | 可每天多篇 | 建议每天1篇 |
| 涨粉方式 | 爆款单篇 | 持续输出 |

## 三、每日创作流程

### 4:00-5:00 学习阶段

1. 分析2-3篇新爆款（小红书+公众号）
2. 记录标题模板
3. 总结内容结构
4. 提取可复用技巧

### 5:00-6:00 创作阶段

1. 根据学习成果创作
2. 小红书：1-2篇（短平快）
3. 公众号：1篇（深度文）

### 6:00 发布阶段

根据内容特性和数据，选择发布时间：
- 早9点/中午12点/晚6点
