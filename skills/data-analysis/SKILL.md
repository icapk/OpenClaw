## 三、爆款规律分析

### 内容角度分析

- 选题是否蹭热点？
- 标题是否有吸引力？
- 封面是否吸引眼球？
- 内容是否有价值？
- 结尾是否有引导？

### 互动数据分析

```
点赞率 = 点赞数 / 阅读量 × 100%
收藏率 = 收藏数 / 阅读量 × 100%
评论率 = 评论数 / 阅读量 × 100%

优质内容：
- 点赞率 > 5%
- 收藏率 > 10%
- 评论率 > 2%
```

### 时间维度分析

- 发布时间
- 发布后2小时数据
- 发布后24小时数据
- 72小时后的自然衰减

## 四、数据采集方法

### 小红书数据

1. 手动记录发布后的各项数据
2. 定期检查数据变化
3. 记录峰值时间

### 公众号数据

1. 从公众号后台导出
2. 记录每篇文章数据
3. 对比发布时间和效果

---

## 五、复盘流程

### 每日复盘

1. 检查前一天的发布数据
2. 记录各项指标
3. 和历史数据对比
4. 发现异常点

### 每周复盘

1. 汇总本周所有文章数据
2. 找出数据最好和最差的内容
3. 分析原因
4. 制定下周优化计划

### 每月复盘

1. 汇总本月数据
2. 分析趋势
3. 总结规律
4. 更新选题库
