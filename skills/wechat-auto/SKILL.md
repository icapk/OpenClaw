# 公众号自动化运营 Skill（v1.0）

> 版本：v1.0 | 更新日期：2026-03-29
> 定位：深度内容，面向职场人/创业者/知识分子

---

## 公众号特点

| 维度 | 特点 |
|------|------|
| 受众 | 职场人、创业者、知识分子 |
| 年龄 | 25-45岁为主 |
| 阅读场景 | 通勤、午休、睡前 |
| 内容偏好 | 深度、思考、独特观点 |

---

## 爆款公式

```
独特观点 + 深度论证 + 金句收尾 + 优质排版
```

---

## 标题技巧

### 公众号标题特点

- 不限字数（建议12-30字）
- 重观点，不重悬念
- 可以更长、更完整

### 标题模板

| 类型 | 公式 | 示例 |
|------|------|------|
| 观点型 | XX的真相 | AI写作的真相 |
| 数字型 | X个XX | 5个让你更值钱的习惯 |
| 疑问型 | 如何XX？ | 如何找到真正适合自己的工作？ |
| 故事型 | 从XX到XX | 从月薪5000到副业过万 |
| 金句型 | XX才是XX | 深度思考才是真正的竞争力 |

---

## 正文结构

### 深度文模板

```
1. 开头（痛点切入）
   - 描述一个普遍问题
   - 引发共鸣

2. 核心观点
   - 亮出独特观点
   - 不人云亦云

3. 深度论证
   - 2-3个分论点
   - 每个分论点有案例/数据支撑

4. 行动建议
   - 具体可操作的建议
   - 不超过3条

5. 金句结尾
   - 一句话总结核心
   - 让人记住
```

### 字数要求

| 类型 | 字数 |
|------|------|
| 短文 | 800-1200字 |
| 标准文 | 1200-2000字 |
| 长文 | 2000-3000字 |

---

## 排版规范

### 基础排版

- 段间距：1.75倍行高
- 正文字号：15-16px
- 标题字号：18-22px
- 两端缩进：8-16px

### 段落规范

- 每段不超过5行
- 重点句子加粗
- 适当引用
- 图文并茂

### 配图规范

- 用公众号AI配图
- 图片和内容相关
- 封面图简洁大气

## 与小红的核心区别

| 维度 | 公众号 | 小红书 |
|------|--------|--------|
| 内容长度 | 1000-3000字 | 300-800字 |
