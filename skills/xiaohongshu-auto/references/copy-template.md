# 小红书文案模板

## 爆款标题公式

### 公式1：数字+痛点+解决方案
> "3个技巧，让你的效率提升10倍"

### 公式2：情绪词+反差
> "姐妹们！这个AI工具真的绝了🔥"

### 公式3：疑问句+好奇心
> "为什么你用的AI工具总是不好用？"

### 公式4：身份认同+价值
> "作为产品经理，我每天用它省下3小时"

## 正文开头模板

### 闺蜜聊天式开头
```
姐妹们！发现一个超好用的[产品/工具]...

你们有没有这种感觉，[描述痛点]？

今天必须给你们安利这个！
```

### 场景代入开头
```
作为一个每天要和[工作内容]打交道的人...

我真的不允许你们不知道这个[工具/方法]！

用了之后感觉[效果描述]
```

## 正文结构模板

```
❶ [第一点/第一个技巧]
   详细说明 + emoji

❷ [第二点/第二个技巧]
   详细说明 + emoji

❸ [第三点/第三个技巧]
   详细说明 + emoji
```

## 结尾互动模板

```
...

你们觉得呢？
评论区告诉我你们的想法~

觉得有用的姐妹记得收藏！

#标签1 #标签2 #标签3
```

## 标签公式

### 常用标签类型
- 领域词：#AI工具 #效率神器
- 身份词：#产品经理 #打工人
- 场景词：#办公好物 #职场干货
- 情绪词：#必备 #超好用 #绝了

### 标签数量
- 精准标签：5-8个
- 不要堆砌标签（最多10个）

## 配图要求

### 封面图（必须）
- 比例：3:4 或 16:9
- 风格：强对比、大标题、信息量大
- 文字：不超过5个字

### 内容图
- 信息图：清晰展示关键信息
- 对比图：Before/After
- 流程图：步骤清晰

### 尾图
- 引导关注：点关注不迷路
- 引导收藏：收藏备用
- 引导评论：评论区见
