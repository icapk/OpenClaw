## 二、发帖全流程SOP

### 流程总览

```
选题来源 → 素材搜集 → 文案生成 → 配图生成 → 排版输出 → 人工审核 → 发布
```

---

### 步骤1：选题来源

**三大来源：**

| 来源 | 操作方式 | 频率 |
|------|----------|------|
| **热搜榜** | 每天早8点/晚6点查看小红书热搜榜，筛选与AI/职场/效率相关话题 | 每天2次 |
| **竞品账号** | 关注「跟着阿亮学AI」「库森说AI」「EasyClaw」等账号，追踪爆款笔记 | 每天1次 |
| **关键词监控** | 监控「OpenClaw」「AI助手」「效率工具」「龙虾」等关键词下的新帖子 | 持续 |

**选题标准：**
- ✅ 与AI/OpenClaw/效率工具有关联
- ✅ 有情绪钩子（好奇、共鸣、利益相关）
- ✅ 有实用价值（可落地、能操作）
- ❌ 不是纯功能介绍，要有场景感

**输出：** 选中选题 → 保存到 `topics/YYYY-MM-DD.md`

---

### 步骤2：素材搜集

**触发条件：** 选题确定后自动执行

**执行步骤：**
1. 在小红书/微信搜索选题关键词，抓取前5-10篇优质内容
2. 提取核心观点、关键数据、精彩案例
3. 整理素材要点（保存到 `research/{topic}.md`）
4. 标注可借鉴角度（差异化切入点）

**素材要求：**
- 不抄袭，只参考角度和结构
- 提取的数据要核实
- 记录素材来源链接

---

### 步骤3：文案生成

**执行方式：** 使用 `xiaohongshu-auto` skill 自动生成

**文案五要素：**

| 要素 | 要求 | 示例 |
|------|------|------|
| **标题** | 15-20字，有情绪钩子 | 「3个技巧，效率提升10倍🔥」 |
| **开头** | 前30字抓注意力，闺蜜聊天式 | 「姐妹们！这个AI工具真的绝了」 |
| **正文** | ❶❷❸分点，每段不超3行，适当emoji |  |
| **结尾** | 引导互动 | 「你们觉得呢？评论区见~」 |
| **标签** | 5-8个精准tag | #AI工具 #效率神器 |

**风格要求：**
- 口语化，像真人聊天
- 避免"AI味"——不用"作为一个人工智能"之类的开场
- 适当使用emoji（1-3个，不要堆砌）

**输出：** 5个标题选项 + 正文 → 保存到 `drafts/{topic}-copy.md`

---

### 步骤4：配图生成

**封面图要求：**

| 类型 | 适用场景 | 尺寸 |
|------|----------|------|
| **大字标题图** | 教程类、技巧类 | 3:4 |
| **场景故事图** | 经验分享、观点输出 | 3:4 |
| **深色代码截图** | 工具对比、技术教程 | 16:9 或 3:4 |

**内容图类型：**
- 信息图：清晰展示关键信息
- 对比图：Before/After 效率对比
- 流程图：步骤拆解

**尾图：**
- 引导关注：「点关注不迷路」
- 引导收藏：「收藏备用」
- 引导评论：「评论区告诉我」

**工具：** 使用 baoyu-image-gen 或其他图片生成工具

**输出：** 保存到 `images/{topic}/`

### 步骤6：人工审核（必须！）

**审核人：** 运营部 / 小麦本人

**审核要点：**

| 检查项 | 要求 |
|--------|------|
| **事实准确性** | 技术描述、数据引用是否正确 |
| **情绪调性** | 是否像真人说话，有无"AI味" |
| **敏感词** | 有无违禁词、过度承诺、虚假宣传 |
| **版权问题** | 图片/素材是否有版权风险 |
| **标签合规** | 标签是否违规或过于泛泛 |
| **排版美观** | 段落、emoji、间距是否舒适 |

**审核后：**
- ✅ 通过 → 移到 `ready/`，准备发布
- ❌ 需修改 → 标注问题 → 返回步骤3修改文案

**发布后：**

- 记录发布时间、标题、标签

## 三、文案写作标准

### 3.1 标题公式

#### 公式1：数字+悬念法
> **结构：** X个/天/人 + 效果悬念
> **示例：**
> - 「3个技巧，效率提升10倍🔥」
> - 「7天搞定AI入门，小白必看！」
> - 「用这个工具，1小时干完1天的活」

#### 公式2：情绪词法
> **结构：** 情绪词 + 感叹
> **示例：**
> - 「绝了🔥这个AI工具我要吹爆！」
> - 「姐妹们！终于有人把OpenClaw讲清楚了」
> - 「救命！用了这个我再也不想加班了」

#### 公式3：身份认同法
> **结构：** 身份标签 + 共鸣点
> **示例：**
> - 「作为PM，我每天用它省下3小时」
> - 「打工人必看！这个AI帮我摸鱼」
> - 「程序员用它自动写代码，太香了」

#### 公式4：对比反差法
> **结构：** 对比A vs B / 前后对比
> **示例：**
> - 「用了2天OpenClaw，我决定放弃Cursor！」
> - 「之前VS之后，效率差距太大了」
> - 「手动做1天 vs AI做1小时」

#### 公式5：蹭热点+实操法
> **结构：** 热点话题 + 实用内容
> **示例：**
> - 「黄仁勋说的AI时代，普通人怎么跟上？」
> - 「蹭个热点：OpenClaw又更新了啥」
> - 「2026年AI Agent元年，你需要知道这些」

作为一个每天要和[工作内容]打交道的人...

用了之后感觉[效果描述]

#### 正文结构

```
❶ [第一点/第一个技巧]
   详细说明 + emoji

   详细说明 + emoji

   详细说明 + emoji

**要求：**
- 每段不超过3行
- 适当使用emoji（❶❷❸🔧🔥✨💡）
- 口语化，像说话

```
