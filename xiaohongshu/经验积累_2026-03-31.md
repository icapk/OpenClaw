## 内容创作模板

### 小红书爆款模板（观点型）
```
开头：姐妹们！+ 热点事件/数据
痛点：我也焦虑过 / 这个问题你肯定遇到过
转折：但我发现一件事——
核心观点：（一句话概括）
分点：123（每点3-5行，带emoji）
金句结尾：别XXX，要XXX + 加油手势
标签：8-10个精准tag
```

### 公众号深度文模板
```
导语：热点背景 + 我的判断
第一部分：现象分析（是什么）
第二部分：原因剖析（为什么）
第三部分：应对策略（怎么做）
结语：金句收尾 + 行动号召
```
