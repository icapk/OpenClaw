# RITUALS.md — 日/周训练清单

## Daily（每天）
- [ ] 30 分钟：阅读高质量源码/技术文档并做 5 行笔记
- [ ] 60 分钟：深度编码（无打断）
- [ ] 每次改动：至少补一个关键路径测试（单测/集成）
- [ ] 提交前：跑一次最小回归并记录结果
- [ ] 收工前：写 5 行复盘（见模板）

### 日复盘模板（5行）
1. 今天完成了什么：
2. 最耗时/最卡点：
3. 根因：
4. 明天可复用改进：
5. 一个可量化指标（如测试通过率/交付时长）：

---

## Weekly（每周）
- [ ] 1 次：架构/设计复盘（选一个真实任务）
- [ ] 1 次：性能或稳定性专项（压测/慢查询/异常链路）
- [ ] 1 次：高质量 Code Review（给出可执行建议）
- [ ] 1 次：整理“可复用模板/脚手架/检查清单”
- [ ] 更新 `memory/skill-gaps.md`（关闭已修复项）

---

## 质量门禁（提交前）
- [ ] 需求满足且边界行为明确
- [ ] 关键回归未破坏
- [ ] 无 P0/P1 阻断
- [ ] 可回滚路径清晰
- [ ] 交付证据齐全
